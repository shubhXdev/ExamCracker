# Shubham Exam Cracker

A high-fidelity "Game Style" Exam preparation app for Indian Government Exams (GK).
**Technology**: Vanilla HTML/CSS/JS (No framework, no build step required).

## How to Play
1. Enter your Codename.
2. Invite friends (Simulated Multiplayer).
3. Answer 100 GK Questions (30s timer each).
4. Score points for speed and accuracy.

## Deployment to GitHub Pages
Since this project uses plain HTML/CSS/JS, it is ready for **GitHub Pages** immediately.

1. **Create a Repository** on GitHub.
2. **Upload files**:
   - `index.html`
   - `css/` folder
   - `js/` folder
   - `assets/` folder (if any)
3. **Go to Settings > Pages**:
   - Source: `Deploy from a branch`
   - Branch: `main` / `root` folder.
4. **Done!** Share the generated URL with friends.

> **Note**: The current version uses **Simulated Multiplayer** (Bots) to allow for immediate gameplay without a backend server. If you deploy this, players will see the same questions but their leaderboards will be independent (Local Only).
